# InkMind 📖✨

Application de roleplay immersif.

## Mettre à jour l'app

1. Modifie `index.html` sur GitHub
2. Commit → la build se lance automatiquement
3. Va dans **Actions** → clique sur la dernière build → télécharge **InkMind-APK**
4. Installe l'APK sur ton téléphone

## Première installation

Active "Sources inconnues" dans les paramètres Android avant d'installer l'APK.
